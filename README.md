# Slat Spacing Calculator

![Slat Spacing Calculator](static/icon-512.png)

A simple calculator for determining optimal slat spacing and positioning for fences, balustrades, etc.

## Features

- **Max Spacing Mode** - Maximize gaps between slats without exceeding a specified maximum
- **Min Spacing Mode** - Fit as many slats as possible while maintaining a minimum gap
- **Number of Slats Mode** - Evenly distribute a specific number of slats

Display slat positions by left edge, center, or right edge for easy measuring during installation.

Available [online](https://public.patmat.co.nz/slats/index.html)
